When making changes to the repository, please ensure the following:
* Ask confirmation before committing, pushing and/or creating a pull request with the changes. This is to ensure that all changes are reviewed and approved before being merged into the main branch.
* always ensure `Cargo.lock` is up to date and committed to the repository
* ensure `README.md` is up to date with usage instructions and examples
* ensure `AGENTS.md` is up to date with agent descriptions and usage instructions
